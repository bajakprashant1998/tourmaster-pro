# TourMaster Pro

## Project Info

This is a tour management application built with Vite, React, shadcn-ui, and Tailwind CSS.

## Getting Started

To run the project locally:

1.  Clone the repository.
2.  Install dependencies:
    ```sh
    npm install
    ```
3.  Start the development server:
    ```sh
    npm run dev
    ```

## Technologies

-   Vite
-   TypeScript
-   React
-   shadcn-ui
-   Tailwind CSS
